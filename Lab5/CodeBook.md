# Модифікації

* Об'єднано тестовий та навчальний набори даних.
* Виділені лише вимірювання середнього значення та стандартного відхилення величин.
* Назви вимірюваннь приведені до більш зручного вигляду
* Створено другий незалежний акуратний набір даних (tidy dataset) із середнім значенням для кожної змінної для кожної діяльності та кожного суб’єкту (subject).


# Опис акуратного набору даних

## Ідентифікатори
Перші дві колонки - Суб'єкт та Діяльність.
* Суб'єкт: унікальний ідентифікатор суб'єкту.
* Діяльність: назва діяльності, що виконував суб'єкт під час вимірювань
